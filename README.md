# Adaptive Survival

Adaptive Survival is a Minecraft Bedrock Edition add-on prototype focused on dynamic survival progression.

## Core Features
- Dynamic hostile mob scaling
- Progressive survival difficulty
- Resource scarcity system
- Persistent world progression
- Multi-phase survival escalation

## Current Phases

### Phase 1 — Early Survival
Normal survival conditions.

### Phase 2 — Rising Threat
Hostile mobs gain increased speed and strength.

### Phase 3 — Resource Collapse
Food sources become scarcer while hostile pressure increases.

## Design Goals
- Maintain long-term survival tension
- Prevent late-game stagnation
- Encourage adaptive gameplay strategies
- Create layered progression systems

## Technical Features
- Persistent scoreboard progression
- Stackable phase architecture
- Modular function system
- Tick-safe progression handling
